# アプリ概要
電子工作の設計情報、写真等を投稿できるWebサービスです。

このサービスを使用することで、
- 自分の作品を他のユーザーと共有できます。
- これから電子工作をしたい人は、他のユーザーの投稿を見て、テーマ選びの参考にできます。

ゲストログインも可能ですので、ご自由にお試しください。
# 背景
現在、勤務先でハードウェア設計をしていますが、
回路設計にあたって過去の製品に使用されていた電子部品を調べていました。

その時に、部品の使用実績が体系的に整理されておらず、過去のどの製品にどの部品が使用されていたか
ということが非常に分かりづらく、苦労しました。
また、製品と部品を関連付けて表示できるようなシステムが欲しいと感じました。

その経験から転じて、個人的に楽しむような電子工作でも、
作品と部品の関連付け、さらには自分の作品をシェアできる、
いわば「電子工作版クックパッド」のようなプラットフォームがあれば役立ちそうだと感じ、開発を決意しました。
